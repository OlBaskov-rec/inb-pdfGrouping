# Changelog

Все заметные изменения проекта документируются в этом файле.
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект придерживается [семантического версионирования](https://semver.org/lang/ru/).

## [0.1.16] — 2026-06-10

### Исправлено
- Отключение предпросмотра больше не оставляет окно шире, чем было (ширина запоминается
  при включении панели и восстанавливается при выключении).

### Изменено
- Убрана строка сообщений рядом с кнопками «Обработать» / «Без пересечений»: рутинные
  подтверждения больше не выводятся, ошибки показываются в общей области уведомлений (баннер).

## [0.1.15] — 2026-06-10

### Исправлено
- При включённом предпросмотре секция «Диапазоны страниц» больше не наезжает на
  «Метку группы» (минимальные ширины колонок + минимальная ширина окна с учётом панели).
- Включение предпросмотра больше не растягивает окно вниз — панель прокручивается,
  высота окна не меняется.

## [0.1.14] — 2026-06-10

### Исправлено
- Поля ввода страниц больше не «вытягиваются» и не выдают ошибку при временном
  очищении поля (значения стали `decimal?` с проверкой на пустоту).

### Изменено
- Поля номеров страниц получили фиксированную ширину под 4 символа (со стрелками ↑/↓).
- Кнопки «Очистить» (слева) и «👁 Предпросмотр» (справа) разнесены по краям секции.
- Окно больше не растёт вниз при появлении предупреждений — все сообщения выводятся
  в области уведомлений.

## [0.1.13] — 2026-06-05

### Добавлено
- Кнопка «Добавить без пересечения» в предупреждении (обрезает диапазон до свободных страниц)
  с предпросмотром добавляемых страниц.
- Автоподстановка метки «A», если ещё нет ни одной группы.
- Поля ввода страниц: отдельная строка, крупнее, со стрелками ↑/↓ (NumericUpDown).

## [0.1.12] — 2026-06-05

### Изменено
- Пересекающийся диапазон не добавляется без решения пользователя (Добавить ещё раз / Убрать);
  до решения другие добавления заблокированы.
- В режиме «Без пересечений» требуется решить конфликт перед продолжением.
- Выбор нового PDF обнуляет всю предыдущую сессию.
- Отступы кнопок в «Диапазоны страниц»; стрелка ► у «Создать группу на каждый диапазон».

## [0.1.11] — 2026-06-05

### Добавлено
- «+ Добавить диапазон постранично» — выбранные страницы → отдельные 1-страничные диапазоны.
- «Создать группу на каждый диапазон» — каждый диапазон становится отдельной группой.

### Изменено
- Сообщение «Пересечения запрещены» выводится в общей области уведомлений (баннере).

## [0.1.10] — 2026-06-05

### Добавлено
- Разрешение конфликта пересечений: при включении «Без пересечений» с уже имеющимися
  пересечениями — блок предложения непересекающихся диапазонов (обрезка/разбиение,
  приоритет занявшего раньше) с кнопками «Подтвердить» / «Убрать пересекающиеся».
  Алгоритм `PageRangeUtils.ResolveOverlaps` (+тесты).

## [0.1.9] — 2026-06-05

### Изменено
- Баннер предупреждения о пересечении не скрывается после «Добавить ещё раз».
- «Метка группы»: подпись «(любые 15 симв.)» под заголовком; 5 кнопок (A–E) в один ряд;
  «Текущие диапазоны» — под кнопками.

## [0.1.8] — 2026-06-05

### Добавлено
- Кнопки быстрого выбора метки группы (A–F).
- Объединение в существующую группу: при вводе её имени — вопрос «Добавить диапазоны в неё?».
- Метка группы: ограничение 15 символов и проверка на недопустимые в именах файлов символы.

### Изменено
- Увеличен зазор справа под полосу прокрутки.
- К дублируемым страницам добавлены единицы («страница 10» / «страницы 70–90»).
- Убран значок ⚠ у кнопок (предупреждения только в баннере).
- «Текущие диапазоны» — по одному на строке и крупнее.
- В режиме «Без пересечений» запрет распространён на создание/объединение групп.

## [0.1.7] — 2026-06-05

## [0.1.0] — 2026-06-02

Первый релиз.

### Возможности
- Открытие PDF через кнопку «Обзор…» или **drag & drop** в окно.
- Деление документа на **группы по диапазонам страниц** с произвольными метками
  (буква/номер/слово); несколько несвязных диапазонов в одной группе.
- Склейка каждой группы в отдельный PDF-файл одним нажатием «Обработать».
- Список созданных файлов и кнопка «Открыть папку» с результатами.
- Кросс-платформенный интерфейс (Windows; macOS — в планах).

### Технологии
- UI: **Avalonia 12** (.NET 10).
- Работа с PDF: **PdfSharp 6** — без внешних утилит (qpdf не требуется).
- Авто-обновление: **Velopack** (фид — GitHub Releases).
- Все зависимости под свободными лицензиями MIT/Apache-2.0 (без AGPL).

### Известные ограничения
- Сборка пока без цифровой подписи — при первом запуске возможно предупреждение SmartScreen.
- Зашифрованные/повреждённые PDF могут не открыться (выводится понятная ошибка).
