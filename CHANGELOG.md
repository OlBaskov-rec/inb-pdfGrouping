# Changelog

Все заметные изменения проекта документируются в этом файле.
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект придерживается [семантического версионирования](https://semver.org/lang/ru/).

## [0.1.5] — 2026-06-05

### Изменено
- Предупреждение о пересечении в статусной строке отмечается заметным красным значком ⚠
  (вместо серого символа) уже при выборе диапазонов.

## [0.1.4] — 2026-06-05

### Изменено
- Панель предпросмотра: заголовок и миниатюры по центру; между начальной и конечной
  страницей — разделитель с информацией о диапазоне (страниц, «от → до»).
- Слоган в «О программе» переведён на английский.
- Уточнён текст предупреждения о пересечении (добавлено «Диапазоном» и номера дублей).

## [0.1.3] — 2026-06-04

### Изменено
- В предупреждении о пересечении перечисляются конкретные номера дублируемых страниц.
- Увеличены минимальные размеры окна (динамические с учётом панели предпросмотра),
  чтобы интерфейс не сжимался внахлёст.

## [0.1.2] — 2026-06-04

### Добавлено
- Сводка «Дублирующиеся страницы: …» в предупреждении о пересечении (компактный список).
- Тумблер «Без пересечений» у кнопки «Обработать» — запрет добавления пересекающихся диапазонов.

### Исправлено
- Падение при изменении размера окна с показанным предупреждением (цикл привязки).
- Наложение элементов при уменьшении окна (минимальная ширина, динамическая с предпросмотром).

## [0.1.1] — 2026-06-04

### Добавлено
- Предпросмотр миниатюр начальной/конечной страниц диапазона («👁 Предпросмотр») с увеличением («🔍»).
- Окно «ℹ О программе» с версией, авторством и ручной проверкой обновлений.
- Версия программы в заголовке окна.

### Изменено
- Предупреждение о пересечении диапазонов с прежними группами: крупный значок ⚠, каждое
  пересечение отдельной строкой, окно расширяется вниз; кнопки «Добавить ещё раз» / «Убрать».
- Адаптивная компоновка: прокрутка и перенос кнопок вместо наложения при уменьшении окна.

## [0.1.0] — 2026-06-02

Первый релиз.

### Возможности
- Открытие PDF через кнопку «Обзор…» или **drag & drop** в окно.
- Деление документа на **группы по диапазонам страниц** с произвольными метками
  (буква/номер/слово); несколько несвязных диапазонов в одной группе.
- Склейка каждой группы в отдельный PDF-файл одним нажатием «Обработать».
- Список созданных файлов и кнопка «Открыть папку» с результатами.
- Кросс-платформенный интерфейс (Windows; macOS — в планах).

### Технологии
- UI: **Avalonia 12** (.NET 10).
- Работа с PDF: **PdfSharp 6** — без внешних утилит (qpdf не требуется).
- Авто-обновление: **Velopack** (фид — GitHub Releases).
- Все зависимости под свободными лицензиями MIT/Apache-2.0 (без AGPL).

### Известные ограничения
- Сборка пока без цифровой подписи — при первом запуске возможно предупреждение SmartScreen.
- Зашифрованные/повреждённые PDF могут не открыться (выводится понятная ошибка).
