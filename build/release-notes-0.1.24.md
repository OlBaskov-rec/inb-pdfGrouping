# PDF Grouping 0.1.24

### Изменено
- Поля ввода номеров страниц блокируются не только при вопросе о пересечении, но и при
  «Решении конфликта» — пока висит любой вопрос, числа изменить нельзя.

### Исправлено
- При смене языка во время показанного предупреждения/конфликта динамические части
  сообщения (диапазоны, «повторяются страницы …», предлагаемые диапазоны, запрос
  объединения) теперь тоже переводятся на выбранный язык — без иноязычных вкраплений.
