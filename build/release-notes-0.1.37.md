# PDF Grouping 0.1.37

## English

### Changed
- The window now grows automatically to fit its content — exactly by the height needed — when
  extra content appears (e.g. an overlap warning) or the page-ranges list gets taller. It only
  grows as far as the current screen allows; if there is no more room, scrollbars appear as before.
- The page-ranges list can now show up to 12 rows (was 5) before it starts scrolling internally,
  so the window can grow to show more ranges.
- The window height is no longer remembered between launches (it is now content-driven); the
  width is still remembered.

## Русский

### Изменено
- Окно теперь автоматически увеличивается под содержимое — ровно на нужную высоту — когда
  появляются дополнительные элементы (например, предупреждение о пересечении) или растёт список
  диапазонов. Растёт строго в пределах текущего экрана; если места больше нет — включаются
  полосы прокрутки, как и раньше.
- Список диапазонов теперь показывает до 12 строк (было 5) перед внутренней прокруткой, поэтому
  окно может вырасти и показать больше диапазонов.
- Высота окна больше не запоминается между запусками (она определяется содержимым); ширина
  по-прежнему запоминается.
