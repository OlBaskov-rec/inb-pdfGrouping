# PDF Grouping 0.1.16

### Исправлено
- Отключение предпросмотра больше не оставляет окно шире, чем было: при включении
  панели ширина запоминается и восстанавливается обратно при выключении.

### Изменено
- Рядом с кнопками «Обработать» / «Без пересечений» больше нет строки сообщений.
  Рутинные подтверждения (например, «Добавлен диапазон») убраны, а ошибки выводятся
  в общую область уведомлений (баннер над блоком сохранения).
