**PDF Grouping 0.1.2**

### Новое
- В предупреждении о пересечении показывается сводка **«Дублирующиеся страницы: …»**
  (объединённый компактный список конкретных страниц).
- Тумблер **«🚫 Без пересечений»** у кнопки «Обработать»: во включённом состоянии
  нельзя добавить диапазон со страницами, уже выбранными ранее.

### Исправления
- Устранено падение приложения при изменении размера окна с показанным предупреждением.
- Устранено наложение элементов при уменьшении окна (минимальная ширина, в т.ч. больше
  при включённом предпросмотре).
