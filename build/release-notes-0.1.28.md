# PDF Grouping 0.1.28

### Изменено
- Имя запускаемого файла portable-версии теперь совпадает с именем приложения —
  `PdfGrouping.exe` (раньше стаб-лаунчер назывался «PDF Grouping.exe» с пробелом, из-за чего
  в папке оказывались два похожих exe). В самой программе отображаемое имя остаётся «PDF Grouping».

> Для чистой раскладки распакуйте portable-архив в **пустую** папку: в корне будут только
> `PdfGrouping.exe` (запуск), `Update.exe`, папки `current` и `packages`. Ярлык на рабочем
> столе делайте на корневой `PdfGrouping.exe` — он сохраняет имя при обновлениях.
