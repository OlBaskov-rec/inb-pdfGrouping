# PDF Grouping 0.1.38

## English

### Changed
- The page-ranges list (and the adjacent "Current ranges" / "Created groups" panels) are kept at a
  fixed height again (up to ~7 rows, then they scroll internally, as in 0.1.36). They no longer
  stretch the window.
- The window now grows automatically only when a message appears in the message area (the red
  warning zone), so the message becomes visible without scrolling. Growth is still bounded by the
  screen; if there is no room, scrollbars appear as before.

## Русский

### Изменено
- Список диапазонов (и смежные панели «Текущие диапазоны» / «Сформированные группы») снова имеют
  фиксированную высоту (до ~7 строк, дальше внутренняя прокрутка, как в 0.1.36). Они больше не
  растягивают окно.
- Окно теперь авто-увеличивается только при появлении сообщения в зоне сообщений (красная зона
  предупреждения) — чтобы его было видно без прокрутки. Рост по-прежнему ограничен экраном; если
  места нет — включаются полосы прокрутки, как раньше.
