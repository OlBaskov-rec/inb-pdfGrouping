**PDF Grouping 0.1.0** — первый релиз.

Программа делит PDF на группы страниц по заданным диапазонам и склеивает каждую группу
в отдельный файл.

### Возможности
- Открытие PDF кнопкой «Обзор…» или перетаскиванием в окно
- Группы из произвольных диапазонов страниц с метками (А, Б, 1, Часть1…)
- Несколько несвязных диапазонов в одной группе
- Склейка групп в отдельные PDF одним нажатием «Обработать»
- Список результатов и быстрое открытие папки
- Авто-обновление при выходе новых версий

### Установка
- **Установщик:** `PdfGrouping-win-Setup.exe`
- **Portable:** распакуйте `PdfGrouping-win-Portable.zip` и запустите `PDF Grouping.exe`

> Сборка пока не подписана — при первом запуске Windows SmartScreen может показать
> предупреждение («Подробнее» → «Выполнить в любом случае»).
