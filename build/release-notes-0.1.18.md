# PDF Grouping 0.1.18

### Исправлено
- Кнопки «Обработать» и «Без пересечений» закреплены внизу окна (вне прокрутки) —
  при уменьшении окна они больше не пропадают.
- Восстановлен показ сообщений в области уведомлений (блок-сообщения раньше гасились
  сразу после показа).

### Изменено
- Все заголовки в области уведомлений одного размера: «Внимание: пересечение…»,
  «Без пересечения добавится…», «Решение конфликта…».
- После нажатия «Добавить ещё раз» кнопки решения (Добавить ещё раз / без пересечения /
  Убрать) скрываются — предупреждение остаётся только как информация.
- При активном вопросе по добавлению диапазона включение «Без пересечений» не открывает
  второй вопрос, а выводит сообщение «Сначала примите решение по добавлению диапазона».
- Увеличены отступы у кнопок «Добавить диапазон» / «Добавить диапазон постранично»
  (до и между — как у кнопок создания группы).
