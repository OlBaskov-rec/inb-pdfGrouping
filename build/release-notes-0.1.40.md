# PDF Grouping 0.1.40

## English

### Fixed
- Manual window resizing is now predictable: dragging the bottom edge up no longer makes the
  window jump by random amounts. The automatic height adjustment now reacts only when the message
  area appears or disappears, not while you resize — so the bottom edge stops cleanly at the minimum.

## Русский

### Исправлено
- Ручное изменение размера окна стало предсказуемым: при перетаскивании нижнего края окно больше
  не «прыгает» на разное расстояние. Авто-подгонка высоты теперь срабатывает только при появлении
  или исчезновении зоны сообщений, а не во время ресайза — поэтому нижний край чётко
  останавливается на минимуме.
