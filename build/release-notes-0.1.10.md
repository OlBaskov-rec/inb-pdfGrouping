**PDF Grouping 0.1.10**

### Разрешение конфликта пересечений
Если включить режим **«Без пересечений»**, когда уже добавлены пересекающиеся диапазоны,
в области уведомлений появляется блок **«Решение конфликта по пересечению страниц диапазонов»**:
- предлагаются непересекающиеся диапазоны (каждый — на отдельной строке): пересекающийся
  диапазон обрезается/разбивается по уже занятым страницам (приоритет — у занявшего раньше);
- **«Подтвердить»** — применить предложенные диапазоны;
- **«Убрать пересекающиеся»** — оставить только диапазоны, занявшие страницы первыми.
