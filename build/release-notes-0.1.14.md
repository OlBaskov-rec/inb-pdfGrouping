# PDF Grouping 0.1.14

### Исправлено
- Поля ввода страниц больше не «вытягиваются» и не выдают ошибку при временном
  очищении поля (значения стали `decimal?` с проверкой на пустоту).

### Изменено
- Поля номеров страниц получили фиксированную ширину под 4 символа (со стрелками ↑/↓).
- Кнопки «Очистить» (слева) и «👁 Предпросмотр» (справа) разнесены по краям секции
  «Диапазоны страниц».
- Окно больше не растёт вниз при появлении предупреждений — все сообщения выводятся
  в области уведомлений.
