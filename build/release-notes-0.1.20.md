# PDF Grouping 0.1.20

### Исправлено
- Тумблер «Без пересечений» больше не остаётся «нажатым», если его включили при
  неразрешённом вопросе по добавлению диапазона: режим не активируется, выводится
  подсказка, а кнопка возвращается в выключенное состояние.
  Включить режим можно после того, как текущий вопрос решён тремя кнопками
  (Добавить ещё раз / Добавить без пересечения / Убрать).
