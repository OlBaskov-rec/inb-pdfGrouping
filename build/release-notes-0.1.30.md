# PDF Grouping 0.1.30

### Изменено
- Новый, ненавязчивый сценарий обновления:
  1. При автоматическом обнаружении обновления баннер внизу **сразу не выпрыгивает** — только
     подсвечивается зелёным значок «ℹ» (с миганием).
  2. Пользователь открывает меню «ℹ» → там зелёная кнопка **«Обнаружено обновление X»**.
     По нажатию обновление докачивается (если фон уже скачал — мгновенно) и появляется
     баннер внизу с кнопкой «Обновить и перезапустить».
- В баннере обновления добавлена кнопка **«Отложить»** — скрывает предложение, работа
  продолжается; значок «ℹ» остаётся зелёным, обновиться можно позже из меню.
- Тумблер «Без пересечений» блокируется, пока не решён вопрос по пересекающемуся диапазону
  (как и поля ввода страниц).

### Добавлено
- При сохранении, если файл с таким именем уже есть в папке, имя получает индекс по порядку
  («Имя (1).pdf», «Имя (2).pdf»…) — существующие файлы не перезаписываются. Итоговые имена
  видны в списке «Созданные файлы».
