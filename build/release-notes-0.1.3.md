**PDF Grouping 0.1.3**

### Изменения
- В предупреждении о пересечении теперь перечисляются **конкретные номера** дублируемых
  страниц (напр. «8, 9, 10, 11, 12»); при очень большом числе список сворачивается.
- Увеличены минимальные размеры окна, чтобы интерфейс нельзя было сжать до наложения
  элементов (минимум больше при включённой панели предпросмотра).
