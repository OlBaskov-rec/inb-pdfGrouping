# PDF Grouping 0.1.25

### Исправлено
- Проверка обновлений падала с ошибкой «SSL connection could not be established» в сетях,
  где системный прокси перехватывает TLS (подменяет сертификат GitHub). Теперь обновления
  ходят к GitHub **напрямую**, минуя прокси — проверка настоящего сертификата сохраняется,
  поэтому это безопасно. Прямое соединение к GitHub в таких сетях работает.
- Сообщение об ошибке проверки обновлений теперь показывает и вложенные исключения
  (для диагностики).

> Примечание: исправление действует начиная с 0.1.25 — то есть автоматически обновиться
> «через прокси» с более старой версии не получится. Поставьте 0.1.25 вручную
> (Setup.exe / Portable.zip), дальше авто-обновление заработает.
