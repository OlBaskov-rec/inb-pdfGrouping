# PDF Grouping 0.1.15

### Исправлено
- При включённом предпросмотре секция «Диапазоны страниц» больше не наезжает на
  «Метку группы»: колонкам заданы минимальные ширины, а окну — минимальная ширина
  с учётом панели предпросмотра (поля номеров страниц всегда помещаются целиком).
- Включение предпросмотра больше не растягивает окно вниз — панель прокручивается
  внутри своей области, высота окна не меняется.
