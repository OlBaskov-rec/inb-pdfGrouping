# PDF Grouping 0.1.17

### Изменено
- В предупреждении о пересечении в скобках вместо «дубли:» теперь «повторяются:».
- Строка «Без пересечения добавится: …» выделена отступами сверху и снизу и чуть
  крупнее — лучше отделена от деталей пересечения и от кнопок.
- Кнопки в секции «Метка группы» переименованы:
  «Создать группу для вывода» и «Создать группу для вывода по каждому диапазону».
- Эти кнопки перенесены в нижнюю часть секции — списку выбранных диапазонов отведено
  больше места; увеличены отступы вокруг кнопок (×1.5).
