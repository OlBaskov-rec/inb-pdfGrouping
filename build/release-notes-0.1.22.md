# PDF Grouping 0.1.22

### Добавлено
- Многоязычный интерфейс: русский, английский, французский, испанский, немецкий,
  итальянский, китайский, японский, корейский. Переведены все надписи и сообщения.
- Кнопка выбора языка вверху (рядом с «ℹ»): показывает код текущего языка, по нажатию —
  список языков. Переключение происходит «на лету», выбор сохраняется между запусками.
- При первом запуске язык подбирается по системному (если поддерживается), иначе русский.

### Изменено
- Кнопка «О программе» теперь показывает только значок «ℹ» (без подписи).
