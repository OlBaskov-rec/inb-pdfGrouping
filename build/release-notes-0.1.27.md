# PDF Grouping 0.1.27

### Добавлено
- Если при старте найдено обновление, значок «ℹ О программе» подсвечивается зелёным фоном
  (с кратким миганием), а синий значок «i» сохраняется — заметная подсказка, что доступно обновление.

### Исправлено
- Кнопка «Проверить обновление» больше не показывает пугающее сообщение о блокировке файла,
  когда обновление уже скачано фоновой проверкой. Теперь выводится дружелюбное
  «Обновление X скачано. Нажмите „Обновить и перезапустить“», а действие — по зелёной кнопке.
