# PDF Grouping 0.1.33

## English

### Changed
- The "Current ranges" list (Group label panel) and the "Created groups" list are now
  height-limited, like the "Page ranges" list: about 7 rows are shown and then a scrollbar
  appears. The panels no longer stretch far down when there are many ranges or groups.

### Note
- From this release, release notes and the changelog are written in two languages — English
  first, then Russian.

## Русский

### Изменено
- Списки «Текущие диапазоны» (секция «Метка группы») и «Сформированные группы» теперь
  ограничены по высоте, как «Диапазоны страниц»: видно около 7 строк, далее полоса прокрутки.
  Секции больше не «улетают» вниз при большом числе диапазонов или групп.

### Примечание
- С этого релиза заметки и журнал изменений ведутся на двух языках — сначала английский,
  затем русский.
