# PDF Grouping 0.1.32

### Добавлено
- В увеличенном просмотре страницы — кнопки поворота на 90° влево (↺) и вправо (↻),
  расположенные сверху по центру изображения. Поворот только для удобства просмотра
  (на исходный PDF не влияет); при каждом открытии просмотр начинается без поворота.

### Изменено
- Проверка обновлений запускается ещё незаметнее — спустя несколько секунд после открытия
  окна, полностью в фоне (с таймаутом). Старт интерфейса от неё точно не зависит.
