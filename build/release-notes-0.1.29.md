# PDF Grouping 0.1.29

### Изменено
- Минимальная высота окна снижена (660 → 480) для нормальной работы на мониторах с низким
  разрешением. Окно при старте подгоняется под рабочую область экрана, а содержимое при
  нехватке высоты прокручивается; кнопки «Обработать» / «Без пересечений» всегда видны внизу.

### Напоминание
- Увеличить страницу в предпросмотре можно как кнопкой «🔍 Увеличить», так и просто кликом
  по самому изображению страницы (уже работало).
