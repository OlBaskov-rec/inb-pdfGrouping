**PDF Grouping 0.1.7**

### Изменения
- Добавлен зазор справа под вертикальную полосу прокрутки — она больше не наезжает
  на блоки (предупреждение, «Сохранить результаты в», «Сформированные группы», исходник).
- «Диапазоны страниц»: по умолчанию 3 строки; при 4–5 диапазонах блок (и соседние колонки)
  растягиваются, чтобы строки были видны; с 6-й строки — прокрутка внутри списка;
  добавлен отступ до полосы прокрутки.
- Убрано дублирующее сообщение о добавлении пересекающегося диапазона у кнопок
  «Обработать»/«Без пересечений» (предупреждение остаётся в баннере).
- Версия приложения синхронизирована с версией сборки.
